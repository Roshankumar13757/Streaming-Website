# Streaming Platform Frontend

React frontend for the streaming platform backend.

## Features

- User authentication (Login, Register, Logout)
- Video browsing and search
- Video upload
- Video playback
- Comments system
- User profile management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will run on http://localhost:3000

## Environment

Make sure your backend is running on http://localhost:5000

## API Configuration

The API base URL is configured in `src/services/api.js`. Update it if your backend runs on a different port.
